# Отчёт по практическим заданиям

## Цель работы

Закрепить навыки работы с сокетами в Python, научиться строить простые сетевые приложения, в том числе клиент-серверные программы, многопользовательские чаты и веб-серверы.  
Особое внимание уделялось пониманию принципов работы протоколов **UDP**, **TCP** и **HTTP**, а также использованию многопоточности при работе с несколькими клиентами.

## Выполнение

В ходе работы были реализованы пять заданий, каждое из которых постепенно усложняло задачу:

1. Простое взаимодействие клиента и сервера по UDP.
2. Выполнение вычислений (теорема Пифагора) на сервере по запросу клиента через TCP.
3. Минимальный HTTP-сервер, отдающий HTML-страницу.
4. Многопользовательский чат на TCP с использованием потоков.
5. Веб-сервер для работы с GET/POST-запросами и хранением данных в JSON-файле.

Все задания оформлены в отдельных разделах с кодом, результатами и выводами.

## Используемые технологии

- Язык программирования **Python 3**.
- Библиотека **socket** — для работы с сетевыми соединениями.
- Библиотека **threading** — для многопоточности (обслуживание нескольких клиентов).
- Формат **JSON** — для хранения и обмена данными.
- Минимальная работа с HTTP: ручной разбор запросов и формирование ответов.
- Среда разработки: Visual Studio Code (локальный запуск программ).
